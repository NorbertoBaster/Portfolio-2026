# My Front-End Portfolio

This is my personal front-end developer portfolio built with **Next.js**, **TailwindCSS**, and **Three.js**.  
It showcases interactive interfaces, clean layouts, and subtle animations to demonstrate my skills in UI development.

## Features
- Responsive design with attention to layout and typography
- Interactive 3D elements using Three.js
- Smooth animations powered by Framer Motion

## Tech Stack
- Next.js | React | TailwindCSS
- Three.js | @react-three/fiber | @react-three/drei
- Framer Motion | react-typed | tsparticles

## Run Locally
```bash
npm install
npm run dev
